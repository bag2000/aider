Задание: Создай подробный файл README.md для системы бэкапа.

Используй блоки кода Markdown для всех команд терминала и примеров конфигурации!

В файле должны быть следующие разделы:
    Описание: Модульная система резервного копирования на базе Restic.

    Структура файлов:
        main.sh: Точка входа, управление окружением и путями.
        init.conf: Центральный файл конфигурации.
        restic_backup.sh: Модуль создания снимков.
        restic_forget.sh: Модуль очистки по политикам retention.
        .restic_pass: Файл с паролем репозитория (необходимо создать вручную).

    Установка и настройка:
        Описание необходимости установки restic.
        Установка прав на исполнение: chmod +x main.sh restic_backup.sh restic_forget.sh.
        Ограничение прав на файл пароля: chmod 600 .restic_pass.

    Конфигурация: Краткое описание ключевых переменных в init.conf (пути, кеш, политики хранения).

    Автоматизация (Cron): Пример строки для crontab -e для ежедневного запуска.

    Особенности:
        Автоматическое создание папки кеша.
        Логирование всех операций в файл и консоль одновременно через tee.
        Проверка прав root перед запуском.
        Инициализация репозитория (restic init), если он не найден.
